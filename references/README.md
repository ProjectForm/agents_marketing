# Pasta de Referências Visuais

Coloque aqui arquivos de imagem e vídeo que servem como referência de estilo.

O Brand Director usa esta pasta para revisar visualmente as imagens geradas
comparando com o estilo que você aprovou.

## Como adicionar referências:

1. Copie o arquivo para esta pasta com um nome descritivo:
   - `estilo-feed-aprovado.png` — exemplo de post de feed que você aprova
   - `ugc-estilo-referencia.mp4` — exemplo do estilo UGC desejado
   - `paleta-cores.png` — referência exata da identidade visual
   - `carousel-referencia.png` — exemplo de slide de carrossel aprovado

2. Descreva o que cada arquivo representa em `knowledge/referencias.md`
   (na seção "Arquivos de referência disponíveis em references/")

## Formatos aceitos:
- Imagens: .png, .jpg, .jpeg
- Vídeos: .mp4, .mov (para consulta humana — Claude não processa vídeo diretamente)
